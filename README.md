# Cloud Chat Application - CPSC471
**Chat Application Project for CPSC-471 Computer Communication**

**Phase 1:** Develop a locally hosted chat application displaying two separate programs communicating with each other over your computer's local ports
- Developed two python programs, client.py and server.py that communicate via the TCP protocol to exchange messages between two or more users in a chatroom in live time.
- First experience using AI development tools built into an IDE, used OpenAI's Codex feature for development, found and noted areas for improvement for deployment in Phase 2
- Specifically, if the server is not running or the client fails to connect, the program crashes with no error message delivered, on the server side, when a client disconnects the server does not list who disconnected, only the IP and port number of the disconnected client. 

**Phase 2:** Deploy the application to a cloud environment, running the server on the cloud and the client from a local machine
